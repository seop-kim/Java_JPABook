과정 요약
---

| 요약         |
|------------|
| JPQL       |
| Criteria   |
| QueryDSL   |
| 네이티브 SQL   |
| 객체지향 쿼리 심화 |


- JPA는 복잡한 검색 조건을 사용하여 엔티티 객체를 조회할 수 있는 다양한 쿼리 기술을 지원한다.  
`Criteria`나 `QueryDSL`은 `JPQL`을 편리하게 사용하도록 도와주는 기술이며 결과적으로 개발자는 `JPQL`을 필수적으로 알고 있어야 한다.
